Quick Edit Checklist
1. Open index.html and confirm your bio text.
2. Update contact details in the Contact section.
3. Add links to resume, LinkedIn, or portfolio docs if desired.

Publish Fast Options

Option 1: Netlify Drop (fastest)
1. Go to app.netlify.com/drop.
2. Drag the Desktop/personal folder into the browser.
3. Netlify gives you a live URL immediately.
4. Optional: attach a custom domain in Site Settings.

Option 2: GitHub Pages
1. Create a new GitHub repository.
2. Upload the files from Desktop/personal.
3. In repository settings, enable GitHub Pages from main branch root.
4. Your site will be live at username.github.io/repo-name.

Option 3: Custom Domain
1. Buy a domain from Namecheap, Google Domains, or Cloudflare.
2. Point DNS to your Netlify or GitHub Pages site.
3. Use your name-based domain for professional branding.
